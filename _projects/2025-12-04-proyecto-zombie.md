# Proyecto: Modelo Epidemiológico Zombie

Uso del modelo matemático **SIR** (Susceptibles, Infectados, Recuperados) adaptado a un escenario de brote zombie. Se analizan los puntos de equilibrio y la estabilidad del sistema dinámico bajo diferentes tasas de infección.

### Tecnologías usadas:
* **Ecuaciones Diferenciales Ordinarias (EDO)**
* **Python** (Scipy para integración numérica)
* **Matplotlib** para visualización de fases.

[Link al repositorio de este proyecto](https://github.com/JhonatanSolano/ProyectoZombie)
