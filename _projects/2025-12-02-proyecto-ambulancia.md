# Proyecto: Optimización de Rutas de Emergencia

Desarrollo de un modelo de optimización para la asignación eficiente de ambulancias en zonas urbanas, minimizando el tiempo de llegada mediante algoritmos de rutas críticas.

### Tecnologías usadas:
* **Matemática Aplicada** (Investigación Operativa)
* **Python** (NetworkX para grafos)
* **Geopandas** para análisis espacial.

[Link al repositorio de este proyecto](https://github.com/JhonatanSolano/ProyectoAmbulancia)
