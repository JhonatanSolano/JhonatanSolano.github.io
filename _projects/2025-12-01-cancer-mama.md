# Proyecto: Clasificación de Cáncer de Mama

Este proyecto implementa modelos de Machine Learning para el diagnóstico binario (benigno/maligno) analizando métricas de núcleos celulares. Se enfocó en la optimización de hiperparámetros para maximizar el *Recall*, crucial en diagnósticos médicos.

### Tecnologías usadas:
* **Python** (Pandas, Numpy)
* **Scikit-Learn** (SVM, Random Forest)
* **Matplotlib/Seaborn** para análisis exploratorio de datos (EDA).
