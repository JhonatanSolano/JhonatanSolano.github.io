# Proyecto: DataPrep Personas - DANE

Este proyecto se centra en la limpieza, transformación y preparación de grandes volúmenes de datos provenientes de los censos y encuestas del DANE. El objetivo principal es normalizar la información demográfica para facilitar su posterior análisis estadístico o modelado.

### Tecnologías usadas:
* **Python** (Pandas para manipulación de datos)
* **ETL Processes** (Extracción, Transformación y Carga)
* **Jupyter Notebooks** para documentación del proceso.

[Link al repositorio de este proyecto](https://github.com/JhonatanSolano/DataPrep-Personas-DANE)
